# Intro
This project demonstrates a real-world AI agent system that can dynamically switch between multiple LLM providers, use external tools like web search, and interact through a clean UI

# Features (Aligned with code)
- Multi-LLM Support (OpenAI + Groq)
- Tool Calling (Tavily Web Search)
- ReAct Agent (LangGraph)
- FastAPI Backend
- Streamlit UI
- Dynamic Model Selection

# Honest Note
Note: Current version handles single-turn queries. Memory-based conversations can be extended using LangGraph state management.
